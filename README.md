# Go To-Do CLI

A simple command-line to-do list application written in Go. Tasks are saved to a local `tasks.json` file.

## Features

- Add new tasks
- List all tasks
- Mark tasks as complete
- Delete tasks
- Persistent task storage (tasks are saved between runs on a json file)

## Usage

1. **Build the app:**
   ```sh
   make build

2. **Run the app**
    ```sh
    make run
3. **Delete app**
    ```sh
    make clean
